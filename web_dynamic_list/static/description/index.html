Dynamic List with Editable and Export feature.

Arrange any list view on the fly without any technical knowledge.

This module allows you to Arrange list views on the fly and thus offers a quick and easy way to view the desired data. Furthermore the List view can be switched from Readonly mode to Editable mode and the current list can be exported directly into an Excel file.

List of Features


  • It Work With Both Community And Enterprise Version.
  • Listview has advanced options that can save you time and add powerful possibilities into your work.
  • Export current view.
  • Saves Tree/List view per user (Admin can settings for all user).
  • List/Tree view remains changed on page refresh.
  • User can change sequence of Columns.

Easy • Dragable • Personal • Modes • Export
