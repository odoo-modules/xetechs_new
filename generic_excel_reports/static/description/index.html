Features

Dynamic Object report generation.

We can use this module on any object of odoo to print the data with selcted field.We can also configure the template of the Report.

Custom styles in reports.

For a well formated data you have to give attractive style to the data so from this module you can pick any style and color for your company details, header of reports and the total field details.

f you want to Group By some data with specific fields, Now it's possible in excel reports. You can Group By selected fields.

You can select submodel to print it's report.

View well formated large data.

The core functionality of this module is to produce excel report for any object of odoo.You can select the fields you want to display on the excel report and get the data in well formated excel sheets.

If you want to dispaly the total of integer and float fields you can do it by configuring the template and can have the total of fields with specific style.

Company and header details in report.

If you want to dispaly the company details with specific style and colour you can show it by specifying the style for the header and company details in the template.

If you want to dispaly report as per sheet wise, So you can also print report as per sheet.

How to configure excel report template style and the configuration of report details.

From setting >Technical > Generic Excel Report Configure you have to select on which module you want to print excel report and with which template you want to print it.

From setting > Technical > Generic Excel Report > Template Configuration you have to select which template you want to apply on this report. in first tab you can define style of font of company details.

If you want to apply diffrent format for the header you can specify this by this tab.

If you want to apply diffrent format for the footer you can specify this by this tab.

Printing excel report.

Once you define template for the paticular object you can see the action of printing that excel report in actions of that object.

After click on the print action of the related module you can see the wizard from which you can download the excel report.
